## 三、各界面 UI 提示词

### 界面 1：主界面

生成一个手机App界面设计，App名称叫"轨迹工坊"。

这是App的首页，用户可以选择"上传图片"或"选择预设图形"来创建跑步路线。

界面内容：

【顶部区域】
- App名称"轨迹工坊"居中，字体粗大
- 右侧是设置图标（齿轮）

【两大入口卡片（并排，等宽）】

左侧卡片：
- 背景：白色带浅蓝色渐变叠加
- 圆角：24px
- 图标：云朵+上传箭头（大号，蓝绿色）
- 标题："上传图片"（16px粗体）
- 描述文字："用自己的照片或图案"（12px灰色）
- 底部小字：支持JPG/PNG（10px浅灰色）

右侧卡片：
- 背景：白色带浅绿色渐变叠加
- 圆角：24px
- 图标：图形库（正方形+圆形+三角形叠放，蓝绿色）
- 标题："选择图形"（16px粗体）
- 描述文字："使用预设模板快速开始"（12px灰色）
- 底部小字：三角形/圆形/星形等（10px浅灰色）

【快速开始区域】
- 区域标题："热门模板"（14px粗体，深灰色）
- 标题右侧有"查看全部"链接（12px蓝绿色）
- 一行横向滚动的迷你图形卡片，每个卡片：
  * 圆形背景（直径44px，纯色）
  * 白色图标居中
  * 下方显示名称（10px）和公里数（9px浅灰色）
- 显示5个图形：圆形、三角形、五角星、心形、正方形
- 最右侧是一个加号图标卡片，写着"更多"

【底部区域】
- 最近使用记录：一行显示一个历史轨迹
  * 左侧：小图标（小猫头像）
  * 中间："小猫跑"（14px）
  * 右侧："5km"（12px蓝绿色）
- 底部导航栏：三个图标
  * 首页（填充，蓝绿色高亮）
  * 我的轨迹（线性）
  * 个人中心（线性）

设计风格：
- 白色背景（#FFFFFF）
- 卡片有轻微阴影（0 4px 12px rgba(0,0,0,0.05)）
- 主色调为蓝绿色渐变（#4FACFE → #00F2FE）
- 圆角统一：大卡片24px，小卡片16px
- iOS风格，无衬线字体（SF Pro/苹方）
- iPhone 15 Pro屏幕比例（9:19）

比例：9:19

### 界面 2：导航界面

生成一个手机App界面设计，App名称叫"轨迹工坊"。

这是实时导航界面，用户正在沿着自定义轨迹跑步。

界面内容：
- 全屏地图视图（浅色地图风格，类似苹果地图）
- 地图上有一条亮橙色的轨迹线，构成一个五角星形状
- 轨迹线为实线，带轻微发光效果
- 轨迹线上有一个蓝色圆点，带有脉冲波纹效果，表示用户当前位置
- 屏幕顶部有一个半透明毛玻璃状态条，显示：
  * 左侧："剩余距离 2.3km"（14px白色粗体）
  * 右侧："偏离 8m"（12px绿色）
- 屏幕底部有一个半透明毛玻璃卡片（圆角24px），上面有三个按钮：
  * 暂停按钮（双竖线图标，白色）
  * 停止按钮（方块图标，白色）
  * 语音开关按钮（喇叭图标，白色，开启状态高亮蓝绿色）
- 屏幕右下角有一个小卡片（圆角16px，半透明黑底）显示：
  * 配速：6:15/km
  * 心率：142
- 地图上路线旁边偶尔出现小箭头提示，写着"直行200m"

设计风格：
- 地图为浅色模式
- 轨迹线亮橙色（#FF6B35），发光
- 底部卡片和顶部状态条采用毛玻璃效果（背景模糊）

- iPhone 15 Pro比例（9:19）
